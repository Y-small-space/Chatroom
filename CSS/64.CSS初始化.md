# CSS初始化

不同浏览器对有些标签的默认值是不同的，为了消除不同浏览器对HTML文本呈现的差异，照顾刘览器的兼容，我们需要对CSS初始化

简单理解：CSS初始化是指重设浏览器的样式。（也称为CSS reset）

每个网页都必须首先进行 CSS初始化。

这里我们以 京东css初始化代码为例。

## Unicode编码字体

把中文字体的名称用相应的Unicode编码来代替，这样就可以有效的避免浏览器解释CSS代码时候出现乱码的问题。

    比如：
    黑体 \9ED1\4F53
    宋体 \5B8B\4F53
    微软雅黑 15FAE\8F6F\96C5\9ED1
