# 1.HTML语法规范

## 1.1 基本语法概述

1. HTML标签是**由尖括号包国的关键词**，例如 \<html>。

2. HTML标签**通常是成对出现的**，例如\<html>和\</html>，我们称为**双标签**。标签对中的第一个标签是开始标签，第二个标签是结束标签。

3. 有些特殊的标签必须是单个标签（极少情况），何如\<br/>，我们称为**单标签**。

## 1.2 标签关系

双标签关系可以分为两类：**包含关系**和**并列关系**。

**包含关系**:

    <head>
        <title></title>
    </head>

**并列关系**:

    <head></head>
    <body></body>
