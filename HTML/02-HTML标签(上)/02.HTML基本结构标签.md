# 2.HTML基本结构标签

## 2.1 第一个HTML网页

每个网页都会有一个基本的结构标签（也称为骨架标签），页面内容也是在这些基本标签上书与，HTML页面也称为 HTML文档

    <html>
        <head>
            <title></title>
        </head>
        <body>
        </body> 
    </html>

每个网页都会有一个基本的结构标签（也称为骨架标签），页面内容也是在这些基本标签上书写。

|标签名|定义|说明|
|:---:|:-:|:--:|
|\<html>\</html>|HTML标签|页面中最大的标签，我们**称为**根标签|
|\<head>\</head>|文档的头部|注意在head标签中我们必须要设量的标签是title|
|\<titile>\</title>|文档的标题|让页面拥有一个属于自己的网页标题|
|\<body>\</body>|文档的主体|元素包含文档的所有内容，页面内容基本都是放到body里面的|
