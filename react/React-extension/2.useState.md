# useState

## 1.基础使用

### 作用

useState为函数组件提供状态 (state)

### 使用步骤

1. 导入 usestate 函数
2. 调用 usestate 函数，并传入状态的初始值
3. 从usestate 函数的返回值中，拿到状态和修改状态的方法
4. 在JSX中展示状态
5. 调用修改状态的方法更新状态

## 2.状态的读取和修改

### 读取状态

该方式提供的状态，是函数内部的局部变量，可以在两数内的任意位置使用

### 修改状态

1. setCount是一个函数，参数表示最新的状态值
2. 调用该函数后，将使用新值替换旧值
3. 修改状态后，由于状态发生变化，会引起视图变化
